5. Which of the following is true about encryption keys? (D7.4, L7.6)

A. Encryption keys must always be stored in a key vault.
marked correct B. Encryption keys must be protected throughout their life cycles.
C. The public key of the sender of a message is used to encrypt the message to achieve message confidentiality.
D. The private key of the sender of a message is used to encrypt the message to achieve message confidentiality.
Feedback

Your answer is correct.

The correct answer is B. An encryption key must be protected throughout its life cycle. Encryption keys may or may not be stored in a key vault, so answer A is incorrect. Answers C and D are structured to imply usage of asymmetric algorithms, in which case confidentiality is achieved by encrypting the message with the public key of the message recipient (not the sender’s keys).

Incorrect answer feedback: A. While it is a good practice to store encryption keys securely, it is not a requirement that encryption keys must always be stored in a key vault. Key vaults provide a secure way to manage and store keys, but there may be other secure storage mechanisms available depending on the specific requirements and context. C. This option is incorrect because it implies the use of asymmetric encryption algorithms where the public key of the recipient is used to encrypt the message, not the sender's public key. Asymmetric encryption is typically used for key exchange and digital signatures, not for message confidentiality. D. This option is incorrect because it suggests that the sender's private key is used to encrypt the message. In asymmetric encryption, the private key is used for decryption, not encryption. The private key is kept secret and is used by the recipient to decrypt the message encrypted with their public key.
