Question 1

18. Which of the following statements BEST represents Security Requirements Traceability Matrix (SRTM)? (D3.7, L3.18)

marked correct A. SRTM can be used for forward, backward, and unilateral requirements traceability. 
B. SRTM can only be used for forward or backward requirement traceability.
C. SRTM guidance is normally stored in a relational database.
D. SRTM is a matrix documenting system security gaps.
Feedback

Your answer is correct.

The correct answer is A. SRTM can be used for forward, backward, and unilateral requirements traceability. Forward traceability maps requirements to test cases. Backward or reverse traceability maps test cases to requirements. Bidirectional traceability maps requirements to test cases and vice versa. The system’s agreedupon security requirements are derived from all sources and are stored in a matrix, commonly in Excel.

Incorrect answer feedback: SRTM can only be used for forward or backward requirement traceability. SRTM guidance is normally stored in a relational database. SRTM is a matrix that documents system security gaps.
